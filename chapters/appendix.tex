\chapter{ตัวอย่างชุดคำสั่ง (Sample Prompts)}


เนื่องจากงานของคุณมีถึง 5 ระดับภาษาไทย ควรโชว์ตัวอย่างให้เห็นชัดเจน:
\begin{itemize}
    \item ตัวอย่าง Prompt DEA/JA: โชว์เทียบกันเลยว่า ภาษาระดับพิธีการ vs ระดับกันเอง หน้าตาต่างกันอย่างไร
    \item ตัวอย่าง Defensive Prompting: รายละเอียดของ System Prompt ทั้ง 12 รูปแบบที่ทีมออกแบบไว้
\end{itemize}

\chapter{ซอร์สโค้ดของโครงงาน (Source Code)}
Scrubbing Function: โค้ดที่ใช้ Regex และ NER ในการกรองข้อมูล

Metric Calculation: โค้ดที่ใช้คำนวณผลลัพธ์หรือวาดกราฟอัตโนมัติ

GitHub Link: ควรใส่ URL ของ Repository ไว้ในหน้านี้ด้วย

\chapter{ผลการทดสอบเพิ่มเติม (Extended Results)}

ตารางสรุปผลละเอียด: ค่า Accuracy หรือ Success Rate ของทุกโมเดลในทุกระดับภาษา (3 โมเดล × 6 ภาษา × 2 ประเภทการโจมตี)

รูปภาพตัวอย่าง Log: ภาพหน้าจอ Terminal หรือหน้า Google Sheets ที่ระบบบันทึกข้อมูลไว้
